# **App Name**: Ouro Marketplace

## Core Features:

- Product Catalog: Display a catalog of graphic materials organized by category.
- Online Quote: Allow customers to generate online quotes by selecting product options and uploading artwork.
- WhatsApp Integration: Enable customers to initiate complex orders or inquiries via WhatsApp.
- Customer Dashboard: Provide a dashboard for customers to track their order status and download artwork.
- Admin Panel: Enable administrators to manage products, categories, orders, and update order statuses.
- Firestore Integration: Store product, category, order, and user data in Firestore.

## Style Guidelines:

- Primary color: Orange (#FF6B07) to represent the brand's identity.
- Background color: Desaturated orange (#F2E6DA) for a bright, inviting, yet gentle backdrop.
- Accent color: Yellow (#EA9217) for interactive elements and highlights, creating contrast and focus.
- Body and headline font: 'PT Sans', a humanist sans-serif to provide a modern and accessible feel to the platform.
- Use clean and professional icons representing different product categories and order statuses.
- Implement a clean and modern layout using Bootstrap for responsiveness and visual appeal.
- Add subtle transitions and animations to enhance user experience and provide feedback on interactions.
